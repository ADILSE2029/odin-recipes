# Recipes

A recipe website built while learning HTML and CSS fundamentals through The Odin Project.

## About

This project features a styled homepage with recipe cards linking to individual recipe pages. Each recipe page includes an image, description, ingredients list, and step-by-step instructions.

## What I Practiced

- The CSS box model (padding, border, margin)
- The cascade and specificity
- Block vs inline element behavior
- Building a consistent color scheme and reusable card layout
- Linking a shared stylesheet across multiple pages

## Pages

- Homepage with recipe cards
- Individual recipe pages (ingredients, steps, images)
